# Инструкция по загрузке проекта на Netlify

## Способ 1: Drag & Drop (Самый простой)

1. **Откройте сайт Netlify**
   - Перейдите на https://app.netlify.com
   - Войдите в свой аккаунт

2. **Перейдите в раздел Sites**
   - В верхней панели нажмите на "Sites"
   - Если у вас уже есть проект, выберите его, или создайте новый

3. **Загрузите файлы через Drag & Drop**
   - Найдите раздел "Deploy manually" или "Drag and drop"
   - Перетащите всю папку проекта (или все файлы) в это окно
   - Или нажмите "Browse to upload" и выберите все файлы проекта

4. **Дождитесь деплоя**
   - Netlify автоматически загрузит и опубликует ваш сайт
   - Вы получите ссылку вида: `https://random-name-123.netlify.app`

## Способ 2: Через Git репозиторий (Рекомендуется)

### Шаг 1: Создайте Git репозиторий

1. **Создайте репозиторий на GitHub**
   - Перейдите на https://github.com
   - Нажмите "New repository"
   - Назовите репозиторий (например, "bunker-game")
   - Создайте репозиторий

2. **Загрузите файлы в GitHub**
   
   **Вариант A: Через веб-интерфейс GitHub**
   - В созданном репозитории нажмите "uploading an existing file"
   - Перетащите все файлы проекта
   - Нажмите "Commit changes"

   **Вариант B: Через Git командную строку**
   ```bash
   # Перейдите в папку проекта
   cd "C:\Users\Пётр Дулов\Desktop\Новая папка (3)"
   
   # Инициализируйте Git (если еще не сделано)
   git init
   
   # Добавьте все файлы
   git add .
   
   # Создайте первый коммит
   git commit -m "Initial commit"
   
   # Добавьте удаленный репозиторий (замените URL на ваш)
   git remote add origin https://github.com/ВАШ_ЛОГИН/bunker-game.git
   
   # Загрузите файлы
   git branch -M main
   git push -u origin main
   ```

### Шаг 2: Подключите репозиторий к Netlify

1. **В Netlify Dashboard**
   - Нажмите "Add new site" → "Import an existing project"
   - Выберите "Deploy with GitHub" (или GitLab/Bitbucket)
   - Авторизуйтесь через GitHub
   - Выберите ваш репозиторий

2. **Настройте деплой**
   - Build command: оставьте пустым (или `echo "No build needed"`)
   - Publish directory: `.` (точка)
   - Нажмите "Deploy site"

3. **Автоматические обновления**
   - Теперь при каждом push в GitHub сайт будет автоматически обновляться!

## Способ 3: Через Netlify CLI

1. **Установите Netlify CLI**
   ```bash
   npm install -g netlify-cli
   ```

2. **Войдите в Netlify**
   ```bash
   netlify login
   ```

3. **Инициализируйте проект**
   ```bash
   cd "C:\Users\Пётр Дулов\Desktop\Новая папка (3)"
   netlify init
   ```

4. **Задеплойте**
   ```bash
   netlify deploy --prod
   ```

## Настройка домена (опционально)

1. **В Netlify Dashboard**
   - Выберите ваш сайт
   - Перейдите в "Domain settings"
   - Нажмите "Add custom domain"
   - Введите ваш домен и следуйте инструкциям

## Решение проблем

**Проблема: Сайт не загружается**
- Проверьте, что файл `index.html` находится в корне проекта
- Убедитесь, что все пути к файлам относительные (не абсолютные)

**Проблема: Файлы не обновляются**
- Очистите кеш браузера (Ctrl+Shift+R)
- Проверьте, что файлы действительно загружены в Netlify

**Проблема: Ошибки при деплое**
- Проверьте консоль браузера на ошибки
- Убедитесь, что все файлы (JS, CSS, изображения) загружены
